添加新模块流程:
1. 在 `contents` 目录下创建一个新的 markdown 文件，例如 `hobbies.md`，并添加相应的内容和标题。
2. 在"index.html"文件中body的导航栏和主体部分，找到适当的位置插入新的模块代码块，确保引用正确的 markdown 文件路径。
3. 在"scripts.js"中添加对应的文件列表项，以确保新模块能够被正确加载和显示。

注意事项:
-记得更新的时候记得更新中文和英文两个版本的内容文件。
